# Podcast Admin

Podcast admin tools are available in `/admin/podcasts`.

## Managed content

- Last 10 episodes controls (include/exclude/override)
- Featured guest edit
- Upcoming guest management
- Podcast guest application review
- Podcast sync logs

## Sponsor section

Podcast sponsors use the shared sponsor model and are filtered by `sponsor_scope='podcast'`.

## Page imagery

Podcast hero-band image can be managed via page image manager (`/admin/images`) using:

- `page_key=podcasts`
- `section_key=hero-band`
